# Network Intrusion Detection System (NIDS)

A production-grade, open-source cybersecurity platform that monitors network traffic in real-time, detects malicious or suspicious activities, generates actionable alerts, and provides a professional SOC-style dashboard for security analysts.

Built with **Suricata**, **FastAPI**, and **Next.js 14**.

## Features
- Real-time Traffic Analysis
- SOC Dashboard
- Automated Threat Responses
- Threat Intelligence Enrichment
